Understanding Basic Concepts of Microsoft Azure

Theory

Microsoft Azure is a cloud computing platform by Microsoft that provides services such as compute power, storage, networking, artificial intelligence, analytics, and application hosting. With Azure, users can build, deploy, and manage applications without needing physical hardware.

Azure is widely used because it is scalable, cost-effective, globally available, and supports multiple programming languages and operating systems. It is useful for students, developers, and organizations to host applications, run virtual machines, manage databases, and build AI-based solutions.

The Azure Portal serves as a web-based interface where users can create, configure, and monitor cloud resources. Before accessing it, users must create and verify an Azure account.

Key Concepts of Microsoft Azure

Microsoft Azure Services

Azure provides over 200 cloud services including:

  • Virtual Machines
  • Storage Services
  • Databases
  • Web & Mobile App Hosting
  • AI & Machine Learning Tools
  • DevOps Services
  • Serverless Computing

Azure Portal

A browser-based platform used to manage cloud services. Features include:

  • Custom dashboards
  • Monitoring tools
  • Billing reports
  • Role-based access and resource security

Azure Subscription

A subscription is required to use Azure services. It defines billing and access permissions. New users get a limited free trial with credits.

Resource Groups

A logical container used to organize related resources such as virtual machines, databases, and storage accounts.

Azure Virtual Machines

Cloud-based computers where users choose OS, CPU, RAM, and storage. Used for hosting and developing applications.

Azure Storage

Types include:

  • Blob Storage
  • Disk Storage
  • File Storage
  • Queue Storage

Azure Networking

Includes Virtual Networks, subnets, IP management, and security groups for secure data communication.

Azure App Services

Used to host websites, APIs, and mobile backends with automatic scaling.

Azure SQL Database

A managed relational database service with automatic backup and optimization.

Azure Pricing

Follows a Pay-As-You-Go model where users pay only for the services they consume.

Steps to Create a Microsoft Azure Account

  1. Go to the official Azure website.
  2. Click Start Free.
  3. Sign in or create a Microsoft account.
  4. Verify your phone number.
  5. Add a payment card for identity verification.
  6. Accept terms and finish signup.
  7. Open Azure Portal and explore services such as Resource Groups and Virtual Machines.

Conclusion

Microsoft Azure is a powerful cloud platform that provides tools to build, deploy, and manage applications efficiently. By creating an Azure account and exploring key services like Virtual Machines, Storage, Networking, Databases, and App Services, users can begin working with modern cloud technologies and real-world cloud infrastructure.
