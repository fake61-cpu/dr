Theory

Cryptography is the technique of securing data by converting it into an unreadable form (ciphertext) so only authorized users can access it. It involves encryption (converting plaintext to ciphertext) and decryption (recovering plaintext using a key).

Symmetric Encryption (AES):
Uses the same key for both encryption and decryption. AES (Advanced Encryption Standard) is a fast and secure symmetric algorithm operating on 128-bit data blocks with keys of 128, 192, or 256 bits. It is widely used for file and data encryption.

Asymmetric Encryption (RSA):
Uses a public key for encryption and a private key for decryption. RSA is based on large prime factorization, offering high security but slower performance compared to AES. It is commonly used for secure key exchange and digital communication.
